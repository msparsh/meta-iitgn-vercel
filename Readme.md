# Meta IITGN

Meta IITGN is a modular application built with a **Next.js** frontend and **Express** as API backend, powered by a **Supabase** database.

## 📁 Project Structure

This repository contains the complete codebase, logically separated into the user interface and the API:

* **`frontend/`**: Contains the Next.js user interface, static assets, and client-side dependencies.
* **`backend/`**: Contains the Express server, core business logic, database integrations, and backend tooling.
